## Description
Matlab code for scattering by a semi-infinite elastic plate.
Initially based on the Williams & Porter (2009) model.
(cf JFS2009-paper branch - this should be left as is from now on)

## Installation
Also need another repo of general functions.

* Clone with:
```
git clone https://github.com/tdcwilliams/MatlabLibrary.git
```
* Export path to this repo in .bashrc
```
export MATLAB_LIB_DIR=[path to this repo]
```
